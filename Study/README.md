# 학습 자료 디렉토리 (Study)

이 디렉토리는 코드베이스 학습을 위한 자료들을 관리하는 곳입니다.

## 📚 학습 자료 목록

### 1. `study_guide.md`
- **목적**: 전체 코드베이스 학습 로드맵
- **내용**: 10단계 학습 순서, 각 단계별 이해 목표 및 실습 과제
- **대상**: 신입 개발자 및 프로젝트 참여자

### 2. `launch_file_tutorial.md`
- **목적**: ROS Launch 파일 기초 가이드
- **내용**: `integrated_system.launch` 파일 상세 설명
- **대상**: ROS 초보자

## 📖 학습 순서 권장사항

1. **시작**: `study_guide.md`를 읽고 전체 학습 계획을 파악하세요
2. **2단계**: `launch_file_tutorial.md`를 통해 런치 파일을 학습하세요
3. **3단계 이후**: `study_guide.md`의 다음 단계를 순차적으로 진행하세요
